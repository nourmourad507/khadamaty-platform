# 🏢 منصة خدماتي - Khadamaty Platform

> منصة إدارة أعمال متكاملة لشركة خدماتي

## 📋 نظرة عامة

منصة **خدماتي** هي نظام ERP متكامل يغطي جميع أقسام الشركة:

| القسم | الوصف | الوصول |
|-------|-------|--------|
| 🏠 التسويق العقاري | إدارة العقارات والعملاء | الكل |
| 👷 التوظيف | الوظائف والباحثين عن عمل | الكل |
| 🛠️ الخدمات العامة | طلبات الخدمة | الكل |
| 📦 التوريدات | إدارة الموردين | الكل |
| 📋 **المناقصات** | مناقصات حكومية وخاصة | **المدراء فقط** |
| 🏭 **إدارة المصانع** | الإنتاج والتوريدات | **المدراء فقط** |

---

## 🚀 التشغيل السريع

### المتطلبات
- Node.js 18+
- PostgreSQL 14+ (أو SQLite للتجربة)

### 1. تثبيت التبعيات
```bash
npm install
```

### 2. إعداد البيئة
```bash
cp .env.example .env
# عدل متغيرات البيئة
```

### 3. إعداد قاعدة البيانات
```bash
npm run db:generate
npm run db:migrate
```

### 4. التشغيل
```bash
npm run dev
```

| الخدمة | الرابط |
|--------|--------|
| 🌐 الموقع العام | http://localhost:3000 |
| 🔌 API | http://localhost:3001 |
| ⚙️ لوحة التحكم | http://localhost:3002 |

## 📋 المناقصات (Tenders)

### خاص بالمدراء فقط!

إدارة المناقصات الحكومية والخاصة:
- ✅ إضافة مناقصات حكومية/خاصة/دولية
- ✅ تتبع حالة المناقصة
- ✅ تقديم عروض الأسعار
- ✅ إدارة المستندات
- ✅ إحصائيات المناقصات

### أنواع المناقصات
- **حكومية**: وزارات، محافظات، هيئات
- **خاصة**: شركات، مؤسسات
- **دولية**: منظمات دولية
- **محلية**: مشاريع محلية

---

## 🏭 إدارة المصانع (Factories)

### خاص بالمدراء فقط!

إدارة مصانع الإنتاج والتوريد:
- ✅ إضافة مصانع متعددة
- ✅ إدارة المنتجات والمخزون
- ✅ تتبع المواد الخام
- ✅ أوامر الإنتاج
- ✅ ربط المصانع بالمناقصات

### تبويبات المصنع
1. **نظرة عامة**: إحصائيات وبيانات
2. **المنتجات**: المنتجات والأسعار
3. **المواد الخام**: المخزون والتكلفة
4. **أوامر الإنتاج**: التتبع والجدولة
